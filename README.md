# fc4-tdd

**Repositorio para implementar desafios do curso fc4-tdd**

Este projeto reúne as soluções dos desafios do curso **"fc4-tdd"**. A maior parte do código é desenvolvida em **TypeScript**.

## Pré-requisitos

- [Node.js](https://nodejs.org/) v18.x ou superior
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/) (recomendado)

## Instalação

Clone o repositório e instale as dependências:

```bash
git clone https://github.com/edubrites/fc4-tdd.git
cd fc4-tdd
npm install
# ou
yarn install
```

## Scripts Principais

- **Executar a aplicação:**

  ```bash
  npm start
  # ou
  yarn start
  ```

- **Executar os testes (TDD):**

  ```bash
  npm test
  # ou
  yarn test
  ```

- **Build para produção:**

  ```bash
  npm run build
  # ou
  yarn build
  ```

## Estrutura do Projeto

- **src/** – Código-fonte das soluções
- **tests/** – Testes automatizados (quando disponível)
- **package.json** – Configuração e scripts do projeto

## Dicas

- Use o comando `npm run lint` ou `yarn lint` para verificar problemas de estilização, caso scripts de lint estejam configurados.
- Verifique se há instruções específicas para cada desafio nos diretórios do projeto.
